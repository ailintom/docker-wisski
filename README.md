# USAGE

install [docker](https://docs.docker.com/install/)

and [docker-compose](https://docs.docker.com/compose/install/)

extract wisski-env.tar.gz 

`$ tar -xzvf wisski-env.tar.gz`

start containers

`$ docker-compose up`

go to your browser

`localhost:8081`

quit with 

ctrl + c

and 

`$ docker-compose down`
